# -------------------------------------------------------------------------------------------------
# Native build prerequisites (no sudo? see fallback below)
# -------------------------------------------------------------------------------------------------
# 1) CUDA 12.x toolkit matching PyTorch cu128. On aarch64 with glibc >= 2.39, nvcc 12.0-12.3 chokes
#    on SVE intrinsics in <bits/math-vector.h>; install CUDA 12.4+ (NVIDIA recommends 12.8) and:
#      export CUDA_HOME=/usr/local/cuda-12.8 && export PATH="$CUDA_HOME/bin:$PATH"
# 2) Python development headers (Python.h):
#      sudo apt install python3.12-dev
#    No sudo? Extract the .deb locally and point CPATH at it:
#      cd /tmp && apt download python3.12-dev libpython3.12-dev
#      mkdir -p $HOME/.local/py312-dev
#      dpkg -x libpython3.12-dev_*.deb $HOME/.local/py312-dev/
#      dpkg -x python3.12-dev_*.deb    $HOME/.local/py312-dev/
#    Builds need both .../usr/include (multiarch pyconfig.h) and .../usr/include/python3.12;
#    scripts/build_spconv_local.sh and install_next_steps.sh add both to CPATH automatically.
# 3) When using distro CUDA 12.0/12.1: install gcc-12 g++-12; the scripts set CC/CXX and NVCC -ccbin.
# 4) Kaolin (TRELLIS FlexiCubes optional path): no aarch64+cu128 wheel index is published. The
#    in-tree flexicubes.py uses kaolin.utils.testing.check_tensor only; we ship a permissive
#    fallback so the TRELLIS pipeline imports without kaolin. Install Kaolin from source if
#    you want the full kaolin feature set on aarch64.
